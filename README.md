# SDL-Projects
A repository of my attempts to learn SDL

## Requirements

- C++ compiler toolchain that supports C++ 20
- CMake version 4.2.3

## Building this repository

### Unix / MacOS
  - Create a new folder to clone this repository
  - Clone this repository
    - Ex: `git clone https://github.com/Man-in-a-Kayak/SDL-Projects.git`
  - Navigate into folder
  - Build with CMake
    - `cmake -S . -B build`
  - Choose project you'd like to build with `--target project-name`
    - Ex. `cmake --build build --target project-1`
